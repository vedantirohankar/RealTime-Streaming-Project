\# Real-Time Streaming Project



This project demonstrates a real-time data streaming pipeline using Python, Spark, and Medallion Architecture.



\## Project Architecture



\- Producer generates real-time data

\- Spark processes streaming data

\- Bronze layer stores raw data

\- Silver layer stores cleaned data

\- Gold layer stores aggregated business-ready data



\## Technologies Used



\- Python

\- PySpark

\- Apache Spark

\- Git \& GitHub

\- Medallion Architecture



\## Project Flow



1\. Generate streaming data

2\. Store raw data in Bronze layer

3\. Clean and transform data in Silver layer

4\. Create analytics-ready data in Gold layer
